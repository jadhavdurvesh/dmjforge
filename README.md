- ⚡ Run Python Solutions

##  Installation

Clone Repository

    git clone https://github.com/YOUR_USERNAME/dmjforge.git
    cd dmjforge

Install

    pip install -e .

Launch

    dmjforge

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| R | Run Solution |
| A | Add Test Case |
| X | Exit DMJForge |
